# Hangman-CLI #
GAME INFO
- - - -
This node js application prompts user to guess a randomly generated word.

* Tech Used
    * Node JS
    * Javascript
    * NPM
    * Inquirer

* How to Use
    * Clone this repo
    * Then in command line type: npm install
    * Type: cd Hangman
    * Type: node main.js
    * Follow prompts
